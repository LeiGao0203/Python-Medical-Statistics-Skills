# Tutorial Patterns

Use tutorial notebooks when the goal is to teach a method, workflow, or interpretation pattern.

## Recommended Flow

1. Title and learning goals
2. Conceptual overview
3. Minimal setup and imports
4. Small worked example
5. Guided interpretation
6. Practice variation
7. Common pitfalls
8. Recap and next steps

## Teaching Guidelines

- Prefer small datasets that fit on screen before introducing larger files.
- Introduce one statistical idea at a time.
- Explain why a method is appropriate before showing code.
- Pair each result table or plot with a plain-language interpretation.
- Include checkpoints where learners can predict the next result.

## Medical Statistics Emphasis

- Tie formulas to clinical or epidemiological meaning.
- Name assumptions and failure modes.
- Show how missingness, censoring, or confounding changes interpretation.
- Use realistic variable names and outcome definitions.
