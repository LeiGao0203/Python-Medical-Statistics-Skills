# Contributing

Contributions are welcome, especially new Python medical-statistics skills,
corrections to statistical guidance, reproducible examples, and install/test
improvements.

Before submitting a change:

- Keep each skill focused on one method or workflow.
- Prefer Python-native examples over direct translations of R APIs.
- Include assumptions, diagnostics, and interpretation guidance.
- Keep examples runnable with common Python packages.
- Run `bash -n install.sh tests/install_test.sh`.
- Run `bash tests/install_test.sh`.
- Run notebook scaffold checks if notebook files changed.

Method guidance adapted from external sources must preserve attribution and
license notes.
